This is Tor client compiled to WebAssembly which connects to Tor network (ab)using the SnowFlake WebSocket bridge, plus some glue code to handle HTTP, TLS, and Socks5 protocol in the browser.

Please, try in Chromium. Not working in Firefox 65+, due to some incompatibility with WebSocket bridge server (which I do not control).
